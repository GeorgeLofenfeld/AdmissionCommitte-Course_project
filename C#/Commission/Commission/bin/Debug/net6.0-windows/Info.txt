ИНСТРУКЦИЯ:
1. Для добавления нового абитуриента нажмите кнопку "Добавить заявление";
2. Для просмотра \ редактирования \ удаления заявление необходимо нажать соответствующую кнопку и ввести номер заявления
(чтобы его узнать, необходимо его найти в общем списке на главной странице);
3. Для просмотра проходного балла и изменения количества мест, нажмите соответствующую кнопку;
4. Для формирования отчёта по прошедшим абитуриентам нажмите соответствующую кнопку;
5. Для фильтрации общего списка выберете необходимые критерии в строке после слова "Фильтр";
6. Для выхода из учётной записи нажмите кнопку "Выход";
7. Для пролистывания таблицы при её переполнении прокрутите колесо мыши \ используйте стрелки на клавиатуре \ воспользуйтесь появившимся ползунком справа;
8. Для того, чтобы увидеть тех, кто имеет проходной балл, необходимо в фильтре выбрать специальность и текущий учебный год;

УТОЧНЕНИЯ:
1. Для добавления нового пользователя или изменения списка специальностей обратитесь к администратору;
2. Отчёт сохраняется по пути: ;
3. Средний балл в фильтре необходимо вводить через точку, пример: 4.0, 3.4, и т.д.;

ОШИБКИ И НЕДОРАБОТКИ:
1. Система не учитывает полупроходной балл;
2. Одно заявление может быть подано только на одну специальность, иначе нужно ещё раз вносить абитуриента;
3. Отчёт генерируется в виде текстового файла, а не Word - документа;
4. Отсутствует шифрование в базе данных;
5. Отсутствует возможность добавить дополнительные компетенции абитуреинта;
